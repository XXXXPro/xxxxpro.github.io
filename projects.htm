Мои проекты и разработки

Программные продукты

SiteKnockerBot
— мониторинг доступности сайта. Раз в пять минут бот заходит на указанный URL, проверяет корректность ответа и наличие ключевых слов (если они заданы). Если сервер отвечает некорректно, слишком медленно, выдаёт ошибки PHP или пустую страницу, бот отправляет оповещение в Telegram.

Впервые опубликовано: 2017
Язык: PHP, Go.

Intellect Board Pro
— предназначался для реализации идеи «сайт на основе форума». Помимо обычных форумных разделов с темами и сообщениями может содержать разделы других типов, например, статические страницы, блог или раздел с анонимными вопросами. Самый крупный open-source проект из сделанных мной. На нём работает мой мультиблог 4xpro.ru.

Впервые опубликовано: 2014
Язык: PHP, первая версия — Perl.

VK Private Profiles Highligher
— расширение для броузеров на основе Chromium. Выделяет (обводит красным) ссылки на закрытые профили ВК в дискуссиях, чатах, списках участников групп, списках лайков и т.п. Будет полезно тем, кому часто приходится ходить по чужим профилям ВК, чтобы не тратить время на заход в закрытые профили.

Впервые опубликовано: 2020
Язык: Javascript.

CondiLoader
— библиотека для асинхронной загрузки и инициализации JavaScript-библиотек и стилей CSS для них.

Впервые опубликовано: 2018
Язык: Javascript.

TextCMS
— простая CMS на текстовых файлах. Главная цель создания — максимальная гибкость и возможность произвольного сочетания модулей на одной странице.

Впервые опубликовано: 2008
Язык: PHP.

2Seconds
— онлайн-сервис для упрощения минификации и склейки CSS и JavaScript, а также вставки в CSS мелких изображений с помощью техники data in uri.

Впервые опубликовано: 2017
Язык: Javascript.

TabShulte
— простой генератор для таблиц Шульте с измерением времени прохождения.

Впервые опубликовано: 2014
Язык: Javascript.

TypograferBot
— Telegram-бот для типографирования текстов. Основан на модуле для типографирования, взятом из TextCMS с небольшими доработками.

Впервые опубликовано: 2017
Язык: PHP.

Типологии личности

  • Пирамида типологий — система, позволившая упорядочить типологии личности по тем проявлениям, которые они описывают, и четко провести границы между ними.
  • Информионика — собственная типология личности, описывающая особенности процесса решения жизненных задач. Разработка приостановилась на этапе описания положений отдельных функций.
  • Модель X для соционики и толкование соционики через НЛП.
  • Собственная типировочная методика по соционике — на основе нее я несколько лет занимался типированием на форуме Tipiruem.Ru.
  • Собственные тесты по соционике и ТУАИ.
  • Гибридное типирование — методика, при которой типируемый сочетает развернутые ответы, анализируемые типировщиком вручную с формализованными, анализируемыми автоматически. Создавалась в целях исключения «эффекта Тени», когда типировщик не рассматривает какой-либо тип по субъективным причинам.
  • Социозаметки — описания того, как проявляют себя те или иные соционические признаки или типы в различных жизненных ситуациях, а также разъяснение наиболее распространенных заблуждений соционики.

Нереализованные идеи и концепции

  • Mindmap как средство навигации по сайту.
  • Методика «хвост кометы» для оценки эффективности научной публикации.
  • Способ ввода текста для мобильника — был актуален во времена мобильников с экраном 4″ и меньше. Сейчас, с распространением экранов 5.5″, на которых легко попасть в нужную клавишу, необходимость в нем отпала.
  • Использование клеточных автоматов поверх социального графа для моделирования распространения идей в обществе.
  • Сенсорный экран с распознаванием пальцев.
  • Инфохаб — место сбора всех значимых для пользователя информационных потоков.
  • Система определения типа личности на основе машинного обучения — реализована частично ввиду того, что не хватило надежно определенных типов.
  • Метатипирование — попытка формализовать подход к типированию: создать набор вопросов, на которые должен ответить сам типировщик, разбирая анкету. Не доведена до конца.
